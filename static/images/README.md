# Hướng dẫn thêm hình ảnh cho Motor Việt-Nhật

Thư mục này chứa các hình ảnh cho website. Thêm các file hình sau:

## Hình ảnh cần có:

1. **workshop.jpg** - Hình xưởng sửa chữa của bạn
2. **tools.jpg** - Hình các dụng cụ, thiết bị chuyên nghiệp
3. **mechanic.jpg** - Hình thợ kỹ thuật đang làm việc
4. **bikes.jpg** - Hình các loại xe máy

## Kích thước đề xuất:
- Tỷ lệ: 4:3 hoặc 16:9
- Độ phân giải: Tối thiểu 800x600px
- Định dạng: JPG, PNG

## Hiện tại:
Website đang sử dụng hình ảnh placeholder từ Unsplash. Khi bạn thêm hình ảnh thật vào thư mục này, chúng sẽ tự động thay thế hình placeholder.
